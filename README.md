## Installation
### 1. Requirements
* [Node.js](https://nodejs.org/en/)

### 2. Download
```
git clone https://github.com/TalkLounge/google-doodle-search
cd google-doodle-search
npm install
```

## Usage
### Example
```
node src/index.js --type interactive --desc game
```
